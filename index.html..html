KOTAK HITAM

Saya perlu membuat aplikasi web yang menarik dan fungsional untuk Bumdes Gunung Mulia Desa Giri Mulyo. Aplikasi ini harus fokus pada peternakan bebek petelur dengan 750 ekor bebek. Berikut yang perlu saya pertimbangkan:
  1. Tampilan yang menarik dan profesional untuk website desa
  2. Responsif untuk berbagai perangkat (desktop, mobile, tablet)
  3. Fitur untuk mengubah data dalam aplikasi (mungkin menggunakan localStorage atau form sederhana)
  4. Informasi detail tentang peternakan bebek petelur
  5. Kemungkinan fitur seperti: jumlah telur harian, data kesehatan bebek, laporan produksi
  6. Desain yang menarik dengan warna-warna yang sesuai dengan tema perdesaan/peternakan

Saya akan menggunakan HTML, CSS, dan JavaScript dalam satu file. Untuk styling, saya akan menggunakan Tailwind CSS untuk membuat tampilan yang modern dan responsif.

Mari saya buat aplikasi yang komprehensif dengan beberapa bagian:
  • Header dengan logo/nama Bumdes
  • Dashboard utama dengan statistik
  • Form untuk mengupdate data
  • Informasi tentang peternakan
  • Galeri (placeholder)
  • Footer

Saya akan menggunakan localStorage untuk menyimpan data yang bisa diubah oleh pengguna.
